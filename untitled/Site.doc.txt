The Site Class class represent all prosses of login/logout to the game and operating with a game instanse.

The class provides the following methods :
• login
provide prosses login to the game
• hash
provide prosses of generate a hash value
• register
provide register to the game
• logout
provide prosses logout from the game
• newGame
create new game SESSION
• joinGame
provide prosses join to the new Game
• viewAllGames
provide viewing all created Games
• saveGameZone
provide prosses of save Game
• loadGameZone
provide prosses of load Game
• delGame
provide prosses of delete Game
